# Movie-recommender

> C++ 프로그램이 및 실습 마일스톤 과정을 위한 레포지토리입니다.

## 현재 파일별 용도

1. UserInterface
- 텍스트를 통해 사용자와 상호작용하는 기능들을 가지고 있습니다.
- 선택된 메뉴들을 실행합니다.

2. Movie
- 영화의 정보 및 관련 메소드를 가지고 있습니다.

3. Rating
- 평점의 정보 및 관련 메소드를 가지고 있습니다.

4. User
- 사용자의 정보 및 관련 메소드를 가지고 있습니다.

5. Manager
- Movie, Rating, User 객체들을 보관하고 관리합니다.
- UserInterface의 로직을 위한 기능들을 제공합니다.
